# AI Research Assistant

## Overview

AI Research Assistant is a local LLM-powered application that helps researchers analyze academic papers. Users can upload a research paper PDF, generate summaries, extract key insights, and ask questions about the paper using Retrieval-Augmented Generation (RAG).

## Features

* Upload Research Papers (PDF)
* Automatic PDF Text Extraction
* Research Paper Summarization
* Objective Extraction
* Methodology Extraction
* Key Contributions Identification
* Results Analysis
* Future Work Extraction
* Question Answering using RAG
* Local LLM Integration with Ollama (Gemma 4)
* Interactive Gradio Web Interface

## Tech Stack

* Python
* Gradio
* Ollama
* Gemma 4
* PyPDF2
* Sentence Transformers
* Scikit-Learn
* NumPy

## Architecture

PDF Upload
→ Text Extraction
→ Summary Generation
→ Research Analysis
→ Embedding Generation
→ Similarity Search
→ Question Answering
